SetSSHKey

添加SSH-Key.

1 首先设置你的用户名密码:

git config —global user.email “wangfu0806@126.com”
git config —global user.name “richiewang”

2 生成密钥：

ssh-keygen -t rsa -C “wangfu0806@126.com”

3 输入文件路径：

H:\hexo\blog>ssh-keygen -t rsa -C “wangfu0806@126.com”
Generating public/private rsa key pair. Enter file in which to save the key (//.ssh/id_rsa): H:\git\myssh\ssh
Enter passphrase (empty for no passphrase): Enter same passphrase again:
Your identification has been saved in H:\git\myssh\ssh. Your public key has been saved in H:\git\myssh\ssh.pub.
The key fingerprint is: b0:0c:2e:67:33:ab:c1:50:10:40:0a:ba:c1:80:59:22 wangfu0806@126.com

Note1 有个bug,文件路径中的盘符H必须大写,否则会报错.

Note2 上述命令若执行成功,会在H:\git\myssh目录下生成两个文件id_rsa和id_rsa.pub,最后两步:

4 用文本编辑器打开ssh.pub文件,拷贝其中的内容,将其添加到Add SSH Key

5 将id_rsa和id_rsa.pub拷贝至你Git安装目录下的.ssh目录，>如H:\PortableGit-1.8.4.ssh
